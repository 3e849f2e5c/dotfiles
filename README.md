# dotfiles
My Arch Linux + i3-gaps setup

# Enviroment
- fcitx-skin-hakurei-reimu<sup>[aur](https://aur.archlinux.org/packages/fcitx-skin-hakurei-reimu/)</sup><br>
<sup>Required for the full fcitx skin to work, the files in this git repo are only overrides to make the original Hakurei Reimu work with Mozc</sup>

- i3blocks-contrib<sup>[git](https://github.com/vivien/i3blocks-contrib)</sup><br>
<sup>For a big collection of block scripts</sup>

- fcitx-mozc<sup>[git](https://github.com/3e849f2e5c/i3blocks-mozc)</sup><br>
<sup>My block script for showing fcitx-mozc status</sup>

- rofi<sup>[git](https://github.com/davatorium/rofi)</sup><br>
<sup>Application launcher</sup>

- Theme
  - arc-theme<sup>[git](https://github.com/horst3180/Arc-theme)</sup><br>
  <sup>GTK and QT theme</sup>

  - arc-icon-theme<sup>[git](https://github.com/horst3180/arc-icon-theme)</sup><br>
  <sup>Icon theme</sup>

- zsh
  - oh-my-zsh<sup>[git](https://github.com/ohmyzsh/ohmyzsh)</sup><br>
  <sup>Configuration manager</sup>
  - zsh-autosuggestions<sup>[git](https://github.com/zsh-users/zsh-autosuggestions)</sup><br>
  <sup>For giving ghost suggestions based on your history</sup>
  - zsh-syntax-highlighting<sup>[git](https://github.com/zsh-users/zsh-syntax-highlighting)</sup><br>
  <sup>Syntax colors for your terminal</sup>
  - powerlevel9k<sup>[git](https://github.com/Powerlevel9k/powerlevel9k)</sup><br>
  <sup>Prompt theme</sup>

- ttf-nerd-fonts-hack-complete-git<sup>[aur](https://aur.archlinux.org/packages/ttf-nerd-fonts-hack-complete-git/)</sup><br>
<sup>I use Nerd patched Hack as my main font</sup>
